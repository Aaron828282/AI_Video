# 短视频提示词独立市场语言配置

生成时间：2026-03-06 17:09:21
文件名：_短视频提示词独立市场语言配置.md

## 任务背景
用户要求短视频提示词功能与图词请求功能使用独立的目标市场与语言设置。用户可能先为A市场生成图词，再为B市场生成短视频脚本，因此不能共用同一套表单状态。

## 方案详情

### 现状分析
- 图词请求已具备目标市场/语言下拉+自定义输入。
- 短视频请求尚未接收并透传 locale 参数，且后端提示词中语言约束存在写死为葡语的内容。
- 前端没有短视频专属的市场/语言设置面板。

### 技术方案
1. 前端新增短视频专属 locale 状态与修改回调（与图词状态分离）。
2. 在弹窗中新增“短视频提示词目标市场与语言”第二套表单，支持预设下拉 + 自定义输入。
3. 短视频生成与调试请求统一携带 target_market / prompt_language / in_image_text_language。
4. 后端视频接口解析 locale 参数并透传到模型请求构造。
5. 后端短视频提示词 opening 与 task 改为按 locale 动态生成，移除葡语写死。
6. 在短视频生成结果中保存 requestContext.locale，便于前端回显与复用。

### 影响范围
- web/src/App.tsx
- web/src/types.ts
- server/src/index.js
