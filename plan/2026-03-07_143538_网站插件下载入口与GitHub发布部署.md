# 网站插件下载入口与GitHub发布部署

生成时间：2026-03-07 14:35:38

## 任务背景
用户希望网站中提供浏览器插件下载链接，方便新用户直接获取插件安装包；同时要求将项目推送到 GitHub，并触发 Railway 线上部署更新。

## 方案详情

### 现状分析
- 前端侧边栏已有多个导航入口，可增加下载链接入口。
- 后端当前已托管前端静态资源与图片静态目录，但未开放插件下载目录。
- 项目已连接 GitHub 远程仓库并通过 Railway 自动部署。

### 技术方案
1. 生成插件发布包：将 extension 目录打包为 zip 文件，放入仓库 downloads 目录。
2. 后端新增静态下载目录：通过 /downloads 路径公开插件 zip。
3. 前端添加下载入口：在左侧导航和“使用文档”页面提供可点击下载链接。
4. 文档同步：README 增加插件下载与安装说明。
5. 提交并推送：仅提交本次相关文件，push 到 main 分支触发 Railway 自动部署。
