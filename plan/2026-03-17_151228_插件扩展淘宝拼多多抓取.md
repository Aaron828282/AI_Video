# 插件扩展淘宝拼多多抓取

生成时间：2026-03-17 15:12:28
文件名：2026-03-17_151228_插件扩展淘宝拼多多抓取.md

## 任务背景
现有浏览器插件仅支持 1688 商品抓取，需要扩展至淘宝与拼多多，并保持现有后端与前端流程可用。

## 方案详情

### 现状分析
- 插件仅允许 1688 域名，抓取逻辑强耦合 1688 DOM 结构。
- 服务端入库模型通用，可复用。
- 前端展示文案仍以 1688 为中心。

### 技术方案
- 增加站点识别（1688/淘宝/天猫/拼多多），按站点选择解析器与选择器集合。
- 统一抓取流程保留拖拽交互，结果仍上传到现有 /api/products。
- 服务端增强商品 ID 解析（补充 goods_id）。
- 前端去 1688 化文案。

### 影响范围
- extension/manifest.json
- extension/sidepanel.html
- extension/sidepanel.js
- server/src/index.js
- web/src/App.tsx

### 步骤 1：扩展权限与通用文案
- **操作对象**：extension/manifest.json
- **具体动作**：新增淘宝/天猫/拼多多域名权限，更新名称/描述/标题为通用商品抓取助手
- **预期结果**：插件具备多站点访问权限并呈现通用标题

- **关键里程碑**：否

- **操作对象**：extension/sidepanel.js
